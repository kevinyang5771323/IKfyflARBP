# 学生选课管理系统 java634

## 项目概述

学生选课管理系统是基于SpringBoot和Vue的前后端分离架构实现。系统提供管理员、教师、学生三种角色的功能服务，以实现高效的课程管理和选课流程。

## 技术栈

- **后端技术**：SpringBoot、Mybatis-Plus
- **前端技术**：HTML、CSS、Vue.js、Element-UI、Axios

## 功能模块

### 管理员

- 院系管理
- 专业管理
- 班级管理
- 课程管理
- 选课管理
- 学生管理
- 教师管理
- 管理员管理

### 教师

- 授课查询
- 教师课表
- 成绩录入

### 学生

- 选课
- 已选课程管理
- 课表查询
- 成绩查询
- 信息维护

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- JDK 8
- MySQL 5.7
- IntelliJ IDEA
- Maven
- Node.js

## 部署步骤

1. 创建数据库`course_manager`，并导入SQL脚本。
2. 修改`application.yaml`中的数据库配置信息。
3. 启动后端项目。
4. 在前端项目`CourseManagerVue`目录下，运行以下命令：
   ```
   npm run serve
   ```
   或者
   ```
   yarn run serve
   ```
